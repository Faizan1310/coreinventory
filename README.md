# CoreInventory 📦

A modular Inventory Management System (IMS) built with React.

## Features
- Authentication with OTP-based password reset
- Dashboard with real-time KPIs and stock alerts
- Product management with multi-warehouse stock tracking
- Receipts (incoming stock from vendors)
- Delivery Orders (outgoing stock to customers)
- Internal Transfers between warehouses
- Stock Adjustments with full audit log
- Complete Move History ledger

## How to Run
npm install
npm start
